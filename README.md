
---

## 💻 ページ説明

- **kanri.html**
  - 従業員の管理を行うページ
  -

- **notification.html**
  - 通知を表示するページ

- **settings.html**
  - アプリ全体の設定を行うページ

---

## 🎨 スタイルの分け方

- **common.css**
  - 全ページ共通のスタイル（サイドバー、レイアウトなど）
- **各ページ専用 CSS**
  - ページごとに必要なスタイルのみを記述  

例: `kanri.css` には **従業員管理ページ固有の見た目** を追加

---

## 🚀 使い方

1. プロジェクトフォルダを開く
2. 任意のブラウザで `index.html` を開く
3. サイドバーからページを切り替えて利用
